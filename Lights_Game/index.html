Instructions
This is a single player game against the computer. Once you press 'Start', the computer will make it's first choice and you will see that light turn green, then back to purple. You will then have either 15, 10, or 5 seconds to select the correct light(s), depending on the difficulty you select (EASY, MEDIUM, or HARD). With each round, the computer will select one additional light in the sequence. If you continue to get the sequence correct until the number of rounds you chose to play, then you win.
